Папка для изображений отзывов

Поместите сюда изображения отзывов в форматах:
- .jpg
- .jpeg
- .png
- .gif
- .webp

Изображения будут отображаться в алфавитном порядке.
Пример названий файлов:
- 01_review.jpg
- 02_review.png
- 03_review.jpg

Бот автоматически создаст карусель изображений с навигацией влево/вправо.
